Hello!! Thanks for playing my implementation of the 98point6 drop token!

Here are the instructions for playing the game...

How to Play : 
  - Download 9dt.cc
  - Compile with the command "g++ -Wall -g -std=c++11 -o 9dt 9dt.cc"
  - Run "./9dt" to play!
    The game will begin immediately. 

Added command :
 - HELP : prints the rules of the game
